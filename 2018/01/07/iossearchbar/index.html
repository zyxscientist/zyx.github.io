iOS 组件图鉴 —— Search Bars

Zyxscientist
2018-01-07

Search Bar 可以独立出现，也可以嵌套在 Navigation Bar 或者 content view 里面。如果是在 Navigaiton 之中，那么它可以有吸顶的设计。它可以是默认隐藏的，只有在用户向下滑动的时候才出现。

**Search Bar 与 text field 不同。**不可以用 text field 来代替 Search Bar。

Search Bar 里面要带一个清空按钮。

在有需要（例如模态）的情况下，Search Bar 要带 cancel 按钮。

**假如有必要的话，请带上 placeholder，**以提暗示用户这个 Search Bar 的作用。框内 placeholder 的空间有限，不宜带上太长的说明，如果需要一句话来进行表达，placeholder 应该出现在搜索框之上。

考虑在 Search Bar 被点击，进入到 Search 模态之后，在其下显示一些快捷操作，如历史或者收藏标签等等。
