给设计师的几条良心建议

Zyxscientist
2017-03-21

质量与数量都要有

有一句话非常酷，大家都听过，「质量比数量更重要。」大多数情况下这句话都是对的，但是对于设计这个行当来说可能话就不能这么讲了。

对比起 1). 一个你认为已经是完美的方案出来。2).十个八个但可是每个都一般般的方案。拿出三两个不错的方案出来或许是更好的选择。“质量和数量都要有”。

我们面向需求对产品进行设计，而实际上在你开始设计之前，需求提出者（无论是你的客户、PM又或者是老板）的心目中对于这个产品应该长成什么样子已经有了一个模糊轮廓。面对这种情况，一个方案很难让大家都满意，假如你能多拿出一个方案的话，或许就能减少某些意见上面的分歧。推动项目前进的最好的办法是拿出至少俩方案：

  1. 尽量贴近他们心目中「模糊的轮廓」进行设计（并寻找这种设计会带来的问题）

  2. 作为一个设计师，做出你觉得更优的设计方案

情况可能你在没做出第一个方案之前，就知道这样搞根本行不通，但是把这个「行不通」的方案做出来才是这一步的关键所在。老板们在亲眼所见、亲身体会之前，是无法理解为什么这样做会行不通的。最后的结果通常是，你将方案一的一点点东西加到方案二里面去（毕竟老板），但是你推动项目前进，而且也总好于最后用上了那个行不通的方案一。

设计陈述超重要

设计出一套完美无瑕的解决方案，但你却没能很好地跟别人陈述清楚你的设计思路以及你的设计是如何解决问题的话，这套方案也无法得到别人的赞同。要将自己设计的思想概念植入到别人大脑里面，讲得一手好故事的能力非常重要。

给出俩小建议大家参考参考：

  1. 多做调查研究：多看看同类型产品是怎么做的，比如说友商的竞品，又或者是一些大厂的产品（国外的有 Facebook、Google 以及 Apple 等等，国内有 BAT 等等。）揣摩一下别人遇到同样问题的时候会以什么样的手段来解决，并且尽量去思考为什么他们会这样做，是出于什么样的考量。

  2. 在陈述你的设计的时候尽量把你整个设计的过程都表达出来，本来你是怎么想的，现在为什么会变成这个样子。那么别人就会觉得你这个设计是经过思考的，而不是随意地就蹦了出来，别人也能在一定程度上知道最终呈现在眼前的这个方案是怎么通过一个怎样的过程设计出来的。

假如道理将的不够通，故事说的不够好，那么别人或许并不能完全了解你为什么会这样做，也会怀疑你是否有经过完整全面的思考。要为自己的设计方案进行“辩护”，那么你就要做足够多的功课、准备足够有力的台词。

不要停下学习的脚步

对于设计这个行当来说，停止学习的脚步约等于死亡。拿我自己举个例子吧，「学习」这件小事对于我来说优先级很高，只要是对于一个设计师有益的东西，我都会尝试去或深入或浅表地去了解（各类软件、领导能力、程序语言等等）。

或许你会说你非常忙碌，但是我相信人的一天总能挤出那么一点点的时间来进行学习。并且，不是所有的技能都需要你投入大块的时间来去学习的,有很大一部分看起来很难的东西实际上却要不了你半小时。

一个设计师如果对自己的现状很满意，那他很快就会跟不上时代，

在我的第一份工作里面，我学会了 html，css 和 一些基础的 js，能干一些简单的网页开发了。第二份工作里面，我学会了做插画和动效。而现在，我正在提高自己的表达能力和制作更加精美具体的原型。

以正确的态度面对“潮流”

时髦的用语，新潮的样式，时尚的衣着，什么领域都有属于自己的潮流。而潮流这种东西，绝大部分在经过一段时间之后就会消亡。（现在谁还用 low-poly 可能会被打吧···）分辨出哪些设计潮流会活下去并变为经典，哪些会很快消失对于设计师来说非常重要。

追赶潮流，制作出非常 fancy 的界面能够获得非常多的赞跟粉丝。追波上面有无数多看起来非常悦目但实际环境中并不能使用的设计获得了成百上千的赞。理想很丰满，现实很骨感，我们一定要区分清楚理想与现实。

我们要了解时下最流行的设计，但是千万不要沉迷于此。你天马行空，时髦到 blingbling 的作品在追波上获得了很多的赞，这确实非常棒。但是如果你设计出一个具有实际意义，帮助用户解决了问题的设计，即便只获得了 3 个赞，这也是一个非常棒的设计。

最近面了好几个设计师，当我询问他们关于作品集上面某个作品的设计思路时，他们的言语里总会带上一个又一个时髦的词语，但并没有能够很好地解释一些他们自己做下的设计决定。美观有余，道理不足，很伤。

太长了不想读

不会吧，你真的一条都没读就滚到了最下面了？我明明写得很短啊，好气哦，但是还是总结一下这几个良心建议吧。

  1. 「质量比数量更重要」这句话在设计这个范畴里面并不能很好地成立，作为一个设计师你应该兼顾两者，雨露均沾。

  2. 每个设计你都要能够加以解释，越清晰越好。

  3. 抓紧一切学习新知识的机会。

  4. 摆正心态，不盲目追赶潮流。
