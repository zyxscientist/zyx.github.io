v39.1版本新功能

  1. Fixes Rendering bugs when working in Pixel Mode at high zoom levels
  2. Fixes a bug where inserting new Symbol instances could unexpectedly clip their contents
  3. Fixes a bug where layers with blending would not always display as expected in Mirror
  4. Fixes a possible crasher after exiting text editing
  5. Fixes a bug where text shadow could behave unexpectedly
  6. Fixes a rare bug where layers would refuse to be deselected
  7. Fixes a bug where changing the opacity of Symbol instances could have unexpected results
