v3.4.1版本新功能

Version 3.4.1:

  1. The insides of the Photo Junk module have been completely refactored for improved performance;
  2. Visually improved dark mode of CleanMyMac 3 Menu for better readability;
  3. Added a number of helpful info-tooltips instead of disabled checkboxes;
  4. Reworked CleanMyMac 3 Menu’s preferences to fit all screens better;
  5. Resolved an issue of some leftover files not being detected;
  6. Improved Health monitoring performance when handling severely high RAM usage cases;
  7. Significantly improved Uninstaller’s performance;
  8. Fixed enabling/disabling functionality issues for Safari extensions;
  9. Fixed the disabling feature for hung applications monitoring;
  10. Fixed Reveal in Finder functionality for log files in Uninstaller;
  11. Made sure we’re showing English locale instead of PT-BR for users from Portugal;
  12. Surely, we’ve also fixed all the crashes and exceptions that we knew about.
